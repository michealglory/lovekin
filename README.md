# LoveKin Plugin

Member profiles, courses, assessments, reports, and funding requests for the LoveKin community.

## Local Development
- WordPress: http://localhost/lovekin
- Plugin path: `wp-content/plugins/lovekin`
- Database host for XAMPP: `127.0.0.1`

## Milestone 1 Status
- Plugin bootstrap and activation hook wired
- Custom tables created via `dbDelta()`
- Schema version stored in `lovekin_db_version`
